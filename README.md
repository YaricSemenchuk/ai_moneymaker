# 🤖 MoneyMaker AI-Agent для Telegram

Полностью автоматизированный AI-агент, который находит целевую аудиторию в Telegram и приглашает их выполнять задания на вашей платформе.

## ✨ Возможности

- 🔍 **Автоматический поиск групп** по ключевым словам (заработок, криптовалюта, подработка и т.д.)
- 👂 **Мониторинг сообщений** в целевых группах в реальном времени
- 🧠 **AI анализ** контекста сообщений (использует OpenRouter с бесплатными моделями)
- 💬 **Естественные ответы** - агент отвечает как обычный пользователь, не как бот
- 🛡️ **Защита от банов** - рандомные задержки, соблюдение лимитов, "человеческое" поведение
- 📊 **Полное логирование** всех действий в SQLite БД
- 🌍 **Мультиязычность** - поддержка русского и английского языков

## 📋 Требования

- Python 3.8+
- Telegram аккаунт (с реальным номером телефона)
- API ключ OpenRouter (бесплатный)
- Интернет соединение

## 🚀 Быстрый старт

### Шаг 1: Получить Telegram API данные

1. Перейти на https://my.telegram.org/apps
2. Войти с номером телефона вашего Telegram аккаунта
3. Создать приложение (если нет)
4. Скопировать:
   - **API ID** (первое число)
   - **API Hash** (длинная строка)

### Шаг 2: Получить OpenRouter API ключ

1. Перейти на https://openrouter.ai
2. Зарегистрироваться или войти
3. Перейти в Settings → API Keys
4. Скопировать ключ вида `sk-or-v1-xxxxx`

### Шаг 3: Установить зависимости

```bash
# Перейти в директорию проекта
cd /Volumes/secondary/bot

# Установить зависимости
pip install -r requirements.txt
```

### Шаг 4: Настроить .env файл

```bash
# Копировать пример конфига
cp .env.example .env

# Редактировать .env (использовать ваш редактор)
# Заполнить следующие значения:
# TELEGRAM_API_ID=12345678
# TELEGRAM_API_HASH=abcdef123456
# OPENROUTER_API_KEY=sk-or-v1-xxxxx
```

**Содержимое .env файла:**
```
TELEGRAM_API_ID=<ваш_API_ID>
TELEGRAM_API_HASH=<ваш_API_Hash>
OPENROUTER_API_KEY=sk-or-v1-<ваш_ключ>
OPENROUTER_MODEL=meta-llama/llama-2-7b-chat
REFERRAL_BOT=@moneymakerquest_bot
LOG_LEVEL=INFO
DB_PATH=moneymaker_agent.db
```

### Шаг 5: Запустить агента

```bash
python main.py
```

При первом запуске:
- Агент запросит номер телефона Telegram аккаунта
- Введите номер в формате: +79991234567
- Введите код подтверждения из Telegram
- Агент начнет работать!

## ⚙️ Конфигурация

### Основные параметры в config_agent.py

```python
# Ключевые слова для поиска групп
TARGET_KEYWORDS = [
    "заработок", "криптовалюта", "подработка", 
    "фриланс", "работа онлайн", "удаленка"
]

# Защита от банов
MESSAGE_DELAY_MIN = 60              # Минимальная задержка (сек)
MESSAGE_DELAY_MAX = 300             # Максимальная задержка (сек)

MAX_GROUPS_TO_JOIN_PER_DAY = 10     # Макс вступлений в день

# Реферальная ссылка
REFERRAL_BOT = "@moneymakerquest_bot"
```

## 🛡️ Безопасность и избежание банов

### Что делает система для защиты

1. **Рандомные задержки** между сообщениями (60-300 сек)
2. **Соблюдение лимитов**:
   - Max 5 сообщений в час
   - Max 10 вступлений в день
3. **Рандомное "человеческое" поведение** (паузы, просмотры)
4. **Проверка доступности групп** перед отправкой

### Рекомендации

- ✅ Начните с небольших лимитов и постепенно увеличивайте
- ✅ Используйте один или два аккаунта, а не много
- ✅ Варьируйте время запуска агента
- ✅ Мониторьте работу в первые дни
- ✅ Убедитесь, что аккаунт "разогрет" (использовался нормально)

## 🐛 Troubleshooting

### "AUTH_KEY_INVALID" или "SESSION_REVOKED"

**Проблема:** Ошибка аутентификации при запуске.

**Решение:**
```bash
# Удалить файл сессии
rm agent_session_1.session

# Запустить заново и заново ввести номер телефона
python main.py
```

### "OpenRouter API check failed"

**Проблема:** Не может подключиться к OpenRouter API.

**Решение:**
1. Проверить интернет соединение
2. Проверить что OPENROUTER_API_KEY установлен в .env
3. Проверить что ключ корректен (получить с https://openrouter.ai)

### Агент не отправляет сообщения

**Проблема:** Модули инициализированы, но сообщения не отправляются.

**Решение:**
1. Проверить логи на ошибки
2. Убедиться что агент успешно вошел в группы
3. Проверить что OPENROUTER_API_KEY рабочий
4. Убедиться что аккаунт не забанен (попробовать отправить сообщение вручную)

## 🚀 Продвинутое использование

### Запуск с пользовательскими параметрами

Отредактируйте `config_agent.py` перед запуском:

```python
# Увеличить агрессивность
MAX_MESSAGES_PER_HOUR = 10
MESSAGE_DELAY_MIN = 30
MESSAGE_DELAY_MAX = 120

# Добавить свои ключевые слова
TARGET_KEYWORDS.extend(["ваше слово", "еще слово"])
```

### Использование нескольких аккаунтов

Создайте несколько конфиг-файлов для разных аккаунтов:

```bash
# config_agent_1.py
# config_agent_2.py
# config_agent_3.py

# Запустить несколько агентов:
python main.py --config config_agent_1.py &
python main.py --config config_agent_2.py &
```

### Интеграция с вашей системой

Используйте `TelegramAgent` класс в своем коде:

```python
from main import TelegramAgent

agent = TelegramAgent()
await agent.initialize()
await agent.run(iterations=5)
```

## 📞 Поддержка и улучшения

### Известные ограничения

- Pyrogram не может искать группы напрямую (нужен веб-парсинг или альтернативные методы)
- Обработка сообщений требует подключения @client.on_message() обработчиков
- Сейчас используется in-memory сессия (можно изменить на сохранение на диск)
